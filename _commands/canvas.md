Manage canvas sessions. Uses the excalibrain:canvas skill.

## Usage
- `/excalibrain:canvas` or `/excalibrain:canvas status` — show current canvas session status
- `/excalibrain:canvas new explore` — start new Explore session
- `/excalibrain:canvas new architect` — start new Architect session
- `/excalibrain:canvas new storyboard` — start new Storyboard session
- `/excalibrain:canvas resume` — resume the most recent canvas session

## Modes
- **explore** — iterative, conversational canvas building
- **architect** — comprehensive multi-section system visualization
- **storyboard** — sequential narrative frames
- **wireframe** — UI screen layouts (coming soon)

When invoked, use the excalibrain:canvas skill to handle the request.
