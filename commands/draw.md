Quick draw a diagram. Uses the excalibrain:draw skill.

## Usage
`/excalibrain:draw <description>`

## Examples
- `/excalibrain:draw flowchart of user authentication with OAuth`
- `/excalibrain:draw architecture of a 3-tier web app`
- `/excalibrain:draw sequence diagram for payment processing`
- `/excalibrain:draw mindmap of React ecosystem`

When invoked, use the excalibrain:draw skill to generate the diagram.
